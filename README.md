# mst-prim-kruskal-benchmark
Trabalho desenvolvido para a disciplina de Algoritmos em Grafos, durante a graduação de Ciência da Computação na Universidade Estadual de Maringá. Este repositório armazena o código e os resultados de uma avaliação realizada entre os algoritmos de Prim e Kruskal para o cálculo de uma Árvore Geradora Mínima.
